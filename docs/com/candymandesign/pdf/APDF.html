Classpublic class APDF
InheritanceAPDF flash.events.EventDispatcher

The APDF class enables to create and export PDF documents from Flex. Simple usage example: 

  
  import com.candymandesign.pdf.APDF;
  
  var pdf:APDF = new APDF();
  pdf.addPage();
  pdf.currentPage.addText("Hello world!");
  pdf.export("http://localhost/toPDF/logic/apdf.php");
