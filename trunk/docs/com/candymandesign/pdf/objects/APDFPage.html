addCell()method
public function addCell($text:String, $x:Number = 10, $y:Number = 10, $fontSize:Number = 12, $fontWeight:String = "", $fontType:String = "Arial"):void

Add new cell in the page. It is very similiar with addText method.

Parameters
$text:String — $text - The text.
 
$x:Number (default = 10) — $x - The x position of the text.
 
$y:Number (default = 10) — $y - The y position of the text.
 
$fontSize:Number (default = 12) — $fontSize - The font size.
 
$fontWeight:String (default = "") — $fontWeight - The font weight. If this value is "", it will be set as normal.
 
$fontType:String (default = "Arial") — $fontType - the font type. Usage example: 

   import com.candymandesign.pdf.APDF;
   import com.candymandesign.pdf.objects.APDFont;
   
   var pdf:APDF = new APDF();
   pdf.addPage();
   pdf.currentPage.addCell("Some really smart text in the cell.",0,0, 12, "B", APDFont.COURIER);
   pdf.export("pathTo_APDF_PHP_file");
addImage()method 
public function addImage($source:DisplayObject, $quality:Number = 60, $x:Number = 10, $y:Number = 10, $width:Number = 0, $height:Number = 0):void

Add new image in the page.

Parameters
$source:DisplayObject — $source - DisplayObject you want to include in the pdf page.
 
$quality:Number (default = 60) — $quality - The quality of the image.
 
$x:Number (default = 10) — $x - x position of the image.
 
$y:Number (default = 10) — $y - y position of the image.
 
$width:Number (default = 0) — $width - The width of the image. If this value is 0 and the height value is 0, the width and height would be the actual size of display object.
 
$height:Number (default = 0) — $height - The height of the image. If this value is 0 and the width value is 0 the image size would be the actual size of display object. Usage example: 

   import com.candymandesign.pdf.APDF;
   
   var pdf:APDF = new APDF();
   pdf.addPage();
   pdf.currentPage.addImage(this, 70, 10,10, 90,0); 
   pdf.export("pathTo_APDF_PHP_file");
   
   // if you set the width(height) 
   // and leave the height(width) as 0,
   // height(width) will be automaticly calculated.
